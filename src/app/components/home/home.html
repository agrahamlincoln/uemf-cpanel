Control Panel

The UEMF Control Panel is the one-stop-shop for all things related to the UEMF Website. If you have not yet already, please login to access the major functions of this application.

If you encounter any issues while using the UEMF site or this Control Panel, please submit a ticket and a member of the IT team will assist you.

The control panel provides functionality as follows:
  • Get src links for images and files (for linking and referencing)
  • Upload/Rename/Delete files stored as Website Content
  • Edit existing pages using a rich text editor
  • Manage your user account for this control panel

If you did not intend to be here, or don't know how you got here, just close this tab and carry on.
